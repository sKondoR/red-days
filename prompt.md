Создай мобильное приложение на React Native для отслеживания менструального цикла с следующими функциями:

1. Основной экран с календарем (используй библиотеку react-native-calendars)
2. Возможность отмечать дни менструации в календаре
3. Система уведомлений для предупреждения о наступлении следующих дней
4. Статистика и аналитика цикла
5. Хранение данных локально (AsyncStorage или SQLite)

Технические требования:
- Используй функциональные компоненты с React Hooks
- Для навигации используй React Navigation
- Для уведомлений используй react-native-push-notification
- Для стилизации используй Nativewind
- Поддержка светлой и темной темы

Структура приложения:
- Главный экран с календарем
- Экран добавления/редактирования записей
- Экран настроек уведомлений
- Экран статистики

Дополнительные функции:
- Расчет средней продолжительности цикла
- Прогнозирование следующих дней
- Возможность добавлять заметки к дням
- Экспорт данных

Создай полный код проекта с необходимыми зависимостями и инструкцией по установке.
