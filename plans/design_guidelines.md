# Гайдлайны для дизайна приложения

## 1. Цветовая палитра

### Основные цвета
- **Первичный цвет**: #E91E63 (розовый) - используется для акцентов, кнопок, важных элементов
- **Вторичный цвет**: #9C27B0 (фиолетовый) - используется для второстепенных элементов
- **Цвет фона**: #FFFFFF (белый) для светлой темы, #121212 (темно-серый) для темной темы
- **Цвет поверхности**: #F5F5F5 для светлой темы, #1E1E1E для темной темы
- **Цвет текста**: #212121 для светлой темы, #E0E0E0 для темной темы
- **Цвет вспомогательного текста**: #757575 для светлой темы, #9E9E9E для темной темы

### Цвета для индикации состояния
- **Цвет менструации**: #FF5252 (красный) - для дней менструации
- **Цвет овуляции**: #4CAF50 (зеленый) - для предполагаемых дней овуляции
- **Цвет фолликулярной фазы**: #2196F3 (синий) - для дней фолликулярной фазы
- **Цвет лютеиновой фазы**: #FF9800 (оранжевый) - для дней лютеиновой фазы
- **Цвет ошибки**: #F44336 (красный) - для сообщений об ошибках
- **Цвет успеха**: #4CAF50 (зеленый) - для успешных действий

## 2. Типографика

### Шрифты
- **Основной шрифт**: Inter или системный шрифт платформы (San Francisco для iOS, Roboto для Android)
- **Жирность**: Regular (400), Medium (500), Semi-bold (600), Bold (700)

### Иерархия текста
- **Заголовок 1**: 28px, Semi-bold, для заголовков экранов
- **Заголовок 2**: 24px, Semi-bold, для подзаголовков
- **Заголовок 3**: 20px, Medium, для секций
- **Основной текст**: 16px, Regular, для основного содержимого
- **Вспомогательный текст**: 14px, Regular, для пояснений
- **Мелкий текст**: 12px, Regular, для дополнительной информации

## 3. Компоненты интерфейса

### Кнопки
- **Primary Button**: 
  - Фон: #E91E63 (персиковый)
  - Цвет текста: #FFFFFF (белый)
  - Радиус скругления: 8px
  - Высота: 48px
  - Отступы: 16px по горизонтали
- **Secondary Button**:
  - Фон: прозрачный
  - Цвет текста: #E91E63 (персиковый)
  - Радиус скругления: 8px
  - Высота: 48px
  - Отступы: 16px по горизонтали
- **Outlined Button**:
  - Фон: прозрачный
  - Цвет текста: #E91E63 (персиковый)
  - Цвет обводки: #E91E63 (персиковый)
  - Радиус скругления: 8px
  - Высота: 48px
  - Отступы: 16px по горизонтали

### Карточки
- **Фон**: #FFFFFF для светлой темы, #1E1E1E для темной темы
- **Тень**: 0px 2px 8px rgba(0, 0, 0, 0.1)
- **Радиус скругления**: 12px
- **Отступы**: 16px со всех сторон

### Поля ввода
- **Высота**: 48px
- **Радиус скругления**: 8px
- **Цвет фона**: #FFFFFF для светлой темы, #2D2D2D для темной темы
- **Цвет обводки**: #CCCCCC для светлой темы, #424242 для темной темы
- **Цвет текста**: #212121 для светлой темы, #E0E0E0 для темной темы

### Переключатели и чекбоксы
- **Цвет активного состояния**: #E91E63 (персиковый)
- **Цвет неактивного состояния**: #9E9E9E (серый)
- **Размер**: 20x20px

## 4. Иконки

- Использовать Material Icons или Feather Icons
- Размер стандартных иконок: 24x24px
- Цвет иконок: #212121 для светлой темы, #E0E0E0 для темной темы
- Цвет активных иконок: #E91E63 (персиковый)

## 5. Отступы и сетка

- **Базовый отступ**: 8px
- **Маленький отступ**: 4px
- **Средний отступ**: 16px
- **Большой отступ**: 24px
- **Огромный отступ**: 32px
- **Сетка**: Использовать 8px сетку для согласованности

## 6. Адаптивность

- Приложение должно корректно отображаться на устройствах с разными размерами экрана
- Минимальный размер сенсорного элемента: 44x44px
- Учитывать безопасные области (safe areas) для iPhone X и выше, а также устройств с вырезом

## 7. Темы

### Светлая тема
- Основной фон: #FFFFFF
- Поверхности: #F5F5F5
- Основной текст: #212121
- Вторичный текст: #757575

### Темная тема
- Основной фон: #121212
- Поверхности: #1E1E1E
- Основной текст: #E0E0E0
- Вторичный текст: #9E9E9E

## 8. Анимации и переходы

- Использовать плавные переходы при навигации между экранами
- Длительность анимации: 300ms
- Тип easing: ease-out
- Анимировать появление и исчезновение элементов
- Использовать микроанимации для интерактивных элементов (нажатия кнопок, переключение чекбоксов)

## 9. Специфические элементы для календаря

- Ячейки календаря: квадратные, с равными сторонами
- Активные дни (менструация): помечать точкой или закрашиванием
- Прогнозируемые дни: помечать светлым контуром
- Выбранный день: выделять кольцом или изменением фона
